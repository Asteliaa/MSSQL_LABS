Управление базами данных и файлами 
    1. В установленном по умолчанию экземпляре SQl Server, используя SQL Server Management Studio, создайте БД под названием Test:
        1.1. Сохраните БД в файле testdata_a и выделите для хранения данных в этом файле 4MB;
        1.2. Сконфигурируйте БД Test для автоматического роста размера файла при достижении лимита на 2MB c максимально возможным размером в 10MB.
    2. Создайте новую файловую группу под названием TestFileGroup.
        2.1. Создайте вторичный файл данных (с расширением ndf) БД Test с размером 5MB под названием testdata_b.
        2.2. Сконфигурируйте созданный вторичный файл данных для автоматического роста размера, при достижении лимита, на 2MB c неограниченным размером.
        2.3. Добавьте testdata_b в созданную ранее файловую группу TestFileGroup
    3. В именованном экземпляре SQL Server c использованием  Transact-SQL (см.лекционный материал) создайте БД (с соответствующими атрибутами), содержащую в идентификаторе Ваши инициалы.
    4. В базе данных (п.1-2) создайте произвольную схему базы данных. 
        4.1. Создайте произвольную таблицу TABLE_1 (среда SSMS) и добавьте ее в существующую схему.
        4.2. Создайте произвольную таблицу TABLE_2 (Transact - SQL) с явным указанием созданной ранее файловой группы. (Используйте справочный материал по модулю «Инструкция CREATE TABLE»)
    5. Создайте таблицу (Transact-SQL) в новой схеме, не принадлежащей текущему пользователю.
